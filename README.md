# django基于神经网络的学生学习情况分析系统(论文+PPT)

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=40132

## 技术介绍
python语言+Django框架(后端)+Mysql(数据库)+Vue前端框架

## 开发工具
Pycharm+Navicat(mysql数据库可视化工具)

## 项目视频演示
- [演示视频1](https://graduation-images.oss-cn-beijing.aliyuncs.com/videos/40132/40132_入迷源码-python032-django基于神经网络的学生学习情况分析系统演示录像2025_9cq57jyc.mp4)

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
